# Weather Data Pipeline

Projekt portfolio do GitHuba pokazujący budowę prostego pipeline'u danych z publicznego API.

## Cel projektu
Aplikacja pobiera dane pogodowe z API, przetwarza je w Pythonie, zapisuje do bazy danych i prezentuje w dashboardzie Streamlit.

## Technologie
- Python
- requests
- pandas
- SQL
- baza danych
- Streamlit
